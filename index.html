Navigation

There is a two ways in Quoto to draw a navigation blocks — a vertical navigation mode, and horizontal one, that regulated by nav.navigation-horizontal or nav.navigation-vertical classes. Both of the modes have four fancy themes, a default, a bordered, a light, and the dark. You can use those themes by append a theme class to elements.

Here is their classes: .navigation-light, .navigation-dark, .navigation-bordered, and without any additional class, navigation element will use a default theme.
